More can probably be found in the README somewhere in this page.

In terms of software development, I mostly prefer things at a lower level. Developing websites does not seem as interesting as developing software for microcontrolers, as an example. The hardware I own is not that niche, and I'm not yet sure if whatever I'll be doing will involve it.
